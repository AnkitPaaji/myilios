## Upload structure
Upload the complete folder contents to your GitHub repository exactly like this:

```

```

Do not rename the images. The HTML already links to them using their existing filenames inside the `assets` folder.

## GitHub Pages
1. Create a new repository.
2. Upload all files and the `assets` folder.
3. Go to Settings > Pages.
4. Select branch `main` and root folder `/`.
5. Save and wait for the GitHub Pages link.
